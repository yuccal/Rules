## Providers

以 rule-providers 格式存放 Clash 规则

| 文件名                 | 包含内容                                                              |
| --------------------- | --------------------------------------------------------------------- |
| domainDomestic.yaml   | 自用直连规则 domain 格式                                               |
| ipcidrDomestic.yaml   | 自用直连规则 ipcidr 格式                                               |
| OneDrive.yaml         | OneDrive 规则                                                        |
| Telegram.yaml         | Telegram 规则                                                        |
